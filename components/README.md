# `components/`

UI for the Premium Peptides Lab storefront. Import with `@/components/...`.

| Folder | Contents |
|--------|----------|
| `ui/` | Design-system primitives — `Button`, `Card`, `GlassPanel`, badges, `SectionHeader` (`ui/index.ts`) |
| `layout/` | `StorefrontChrome`, `SiteHeader`, `SiteFooter`, `AnnouncementBar`, `MobileMenu`, `MobileBottomNav`, `ScrollProgress` |
| `sections/` | Homepage blocks: `HeroSection`, `FeaturedProductsSection`, `PillarsSection`, `CategoryShowcaseSection`, `QualityTrustSection`, `TestimonialsSection`, `CtaBannerSection` |
| `home/` | `HomePage` — composes all homepage sections |
| `product/` | `ProductCard`, `ProductGallery`, `ProductPurchaseModule`, `ProductDetailAccordions`, `RelatedProductsSection`, `ProductPageClient` |
| `shared/` | `CatalogImage`, `SectionReveal`, `PageEnter`, `CountUp`, `OrbitingHelix`, `LightningBolt` |
| `seo/` | `JsonLd` script helper |

**Integration:** See `references/project-scaffold.md` and `references/phase-2-architecture.md`.

**Planned folders (not created yet):** `account/`, `checkout/` — cart/checkout shells will live here when wired.
